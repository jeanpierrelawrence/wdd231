Bodinode is an interactive, motion-driven educational platform designed to demystify complex biological systems through character-driven web animations. By transforming abstract physiological mechanisms, metabolic pathways, and hormonal systems into distinct, relatable personalities, the platform provides high-context learners with an intuitive, visually structured framework for mastering human biology.

Target Audience Scenarios

"I am studying for my endocrinology module and keeping track of how multiple hormones interact in a single pathway is overwhelming. How can I visualize the chain reaction without getting lost in a wall of textbook jargon?"

Solution: The platform provides interactive, character-driven animation sequences that visually map feedback loops, showing hormones physically interacting and changing states in real time.

"I need to review the specific physiological steps of cellular signaling during my commute on my smartphone. How can I interact with the animations efficiently on a small touch screen?"

Solution: The user interface features a fluid, mobile-responsive layout built with hardware-accelerated touch targets, allowing users to scrub through animation timelines step by step.

Color Schema

Primary Accent: #BCEEE4 (Cellular Mint). Used for primary buttons, active UI states, and site accent elements.

Secondary Accent: #BCBFEE (Nervous Periwinkle). Used for secondary layout highlights and interactive accent details.

Tertiary Accent: #EECBBC (Hormonal Peach). Used for decorative accent elements and structural contrast variants across the site.

Quaternary Accent: #E8EEBC (Bio-Flora Sage). Used for supplementary highlights and subtle surface accenting.

Neutral Light: #F3F3F3 (Staging Bone). Used for backgrounds, core container fields, and canvas surfaces.

Neutral Dark: #000000 (Black). Used for text, structural headings, and branding typography layout components.

Typography

Poppins

Usage: All Headings and Body Copy.

Reason: Selected for its clean, geometric sans-serif structures and excellent readability across digital screen displays. It maintains a highly structured, modern aesthetic that supports the platform layout by being highly legible and easy to scan.

  • Heading 1: Poppins SemiBold 6rem
  • Heading 2: Poppins Bold 2.5rem
  • Body Text: Poppins Regular 1rem

IBM Plex Mono

Usage: Mono-spaced elements, interactive control layouts, data readouts, and specific UI accent labels.

Reason: As a monospaced typeface, it provides a distinct visual contrast to the primary geometric font. It introduces technical precision to UI layout accent points, highlighting functional elements and metrics clearly.

  • Button: IBM Plex Mono SemiBold 1.25rem
  • Sub-Headings: IBM Plex Mono Regular 1.25rem
  • Info Markers: IBM Plex Mono Regular 0.938rem
